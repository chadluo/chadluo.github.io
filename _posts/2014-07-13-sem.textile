這個寒假一過，離畢業就只有一年了。據說我校是最後一個學期結束之後再隔半年舉辦畢業典禮，換言之穿學士服還是在夏天。

UNSW 的 CS 課程編排還是有點問題。例如 <abbr title="Engineering Design in Computing">2911</abbr> 這種神神叨叨的必修課，OOP 和 Java 一起講倒是無所謂，拿別人寫得課本跟自己寫得一樣教，筆試居然默寫設計模式那一套還手寫 Java 代碼。去年被 <abbr title="Microprocessors and Interfacing">2121</abbr> 搞得印堂發黑 Java 沒認真看居然掛掉了，噁心的是 CS 許多專業課依賴這門水了吧唧的必修課，拖到今年才過所以能選的專業課就很受限制。不過還好 CS 選課自由度高，只規定了學分，至於什麼時候修什麼課，除了依賴關係倒也沒什麼限制。

在認清我不是學霸之後反而沒什麼壓力了，雖然選課時依然以爲自己什麼都能砍下來，但腦子清醒之後還是換成了簡單的課。雖然我校沒必修三大浪漫，但反正我連 HCI 和數據庫都退掉然後又去搞了 OS，去幹 PL 麼隨便啊。

澳洲的問題是如果爲了移民的話，做獨立 iOS 開發者騙不到工作簽證……估計可以通過註冊公司然後僱傭自己的迂迴方式。

受某死掉的知乎用戶影響看了五分鐘 Clojure 然後暈了。去看下學期 PL 的課件發現居然要 Haskell 於是第無數次重新拾起，然後在 List 周圍吐血不止。

給新房東匯訂金出了點岔子導致只好推遲搬家。

後來查日曆，考試之前悉尼電影節明明就有かぐや姫の物語但沒來得及去看，現在倒好，開學后八月份墨爾本電影節也有。雖然只有一場，但如果就爲這個往返一趟墨爾本，我一定是瘋了。
